## Featured Projects

### Gublu - Emotionally Intelligent AI Companion
> Final Year Project · BSc Artificial Intelligence, University of Greenwich (2026)

A full-stack AI web application that detects and models maladaptive coping behaviours using a hybrid multi-model architecture. Combines NLP, fuzzy reasoning, and machine learning into a single behavioural analysis pipeline.

**Results:**
- XGBoost streak-break predictor: **F1 = 0.92 · Recall = 100% · Accuracy = 84.6%** after RandomizedSearchCV optimisation
- **52 tests · 100% pass rate** across unit, integration, API, simulation and ML evaluation
- PSO-optimised Mamdani fuzzy logic distress scoring with validated boundary tuning
- Full GPT-4.1 conversational journaling with rule-based fallback — works without an API key

**Stack:** Python · Flask · React · Next.js · DistilBERT · XGBoost · Fuzzy Logic · PSO · OpenAI API · NumPy · Pandas

[View Repository](https://github.com/mycatgoestomars/gublu)

### Intelligent Oscillation Control - Space Elevator Dynamics

Nonlinear simulation and intelligent control system for stabilising lateral oscillations in a space elevator cable model.

**Results:**
- **~60% reduction** in peak oscillation amplitude vs. baseline
- Settling time cut from **~18s to ~6s** using PSO-tuned fuzzy parameters
- Disturbance rejection, convergence analysis and 3D visualisation

**Stack:** Python · Mamdani Fuzzy Logic · Particle Swarm Optimisation · Numerical Simulation
