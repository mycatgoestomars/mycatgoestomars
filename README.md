## 🌍 Intelligent Oscillation Control System

Developed a nonlinear simulation and intelligent control system for stabilising space elevator cable oscillations.

### 🔹 Features

- Mamdani Fuzzy Logic Controller
- Particle Swarm Optimisation (PSO)
- Nonlinear damping systems
- Disturbance rejection testing
- 3D system visualisation

### 🔹 Results

- ~60% reduction in oscillation amplitude
- Settling time reduced from ~18s → ~6s

### 🔹 Tech Stack

`Python` `Fuzzy Logic` `PSO` `Numerical Simulation` `Control Systems`
